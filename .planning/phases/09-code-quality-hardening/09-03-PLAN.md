2. Find all `requests.get()` or `requests.Session().get()` calls that hit NearBlocks API endpoints. For each:
   - Wrap in a retry loop with exponential backoff:
     ```python
     import random
     import time

     def _nearblocks_request(self, url, params=None, max_retries=5):
         """Make NearBlocks API request with exponential backoff + jitter."""
         for attempt in range(max_retries):
             try:
                 resp = self.session.get(url, params=params, timeout=30)
                 if resp.status_code == 429:
                     wait = (2 ** attempt) + random.uniform(0, 1)
                     logger.warning("NearBlocks 429 rate limited, retry %d/%d in %.1fs",
                                    attempt + 1, max_retries, wait)
                     time.sleep(wait)
                     continue
                 resp.raise_for_status()
                 return resp.json()
             except requests.exceptions.Timeout:
                 wait = (2 ** attempt) + random.uniform(0, 1)
                 logger.warning("NearBlocks timeout, retry %d/%d in %.1fs",
                                attempt + 1, max_retries, wait)
                 time.sleep(wait)
             except requests.exceptions.ConnectionError:
                 wait = (2 ** attempt) + random.uniform(0, 1)
                 logger.warning("NearBlocks connection error, retry %d/%d in %.1fs",
                                attempt + 1, max_retries, wait)
                 time.sleep(wait)
         raise RuntimeError(f"NearBlocks API failed after {max_retries} retries: {url}")
     ```

   - If no shared module exists, add the method to the class that makes the most NearBlocks calls (likely NearFetcher), and import/reuse from other modules
