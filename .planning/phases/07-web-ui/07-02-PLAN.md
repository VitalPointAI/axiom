- `wallets` table (user's registered wallets)
- `transactions` table (for calculating balances)
- `staking_rewards` table (for staking info)

The dashboard should aggregate across all user's wallets to show:
- Total value in USD/CAD
- Holdings by asset (NEAR, ETH, etc.)
- Active staking positions and rewards
