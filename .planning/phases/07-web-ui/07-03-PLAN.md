<objective>
Build the wallet management interface where users can view, add, edit, and sync their crypto wallets across chains.
</objective>

<context>
Wallet data in database:
- address: wallet address
- chain: NEAR, ETH, Polygon, etc.
- label: user-friendly name
- last_synced_at: timestamp
- sync_status: pending, syncing, complete, error

Syncing triggers the indexer to pull latest transactions.
</context>
