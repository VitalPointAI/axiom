<objective>
Set up the Next.js web application with NEAR wallet authentication using near-phantom-auth. Users should be able to sign in with their NEAR wallet and have their session persisted.
</objective>

<context>
near-phantom-auth is published at `@vitalpoint/near-phantom-auth` and provides:
- Passkey-based authentication
- MPC account support
- Implicit NEAR account creation
- React hooks for auth state

The existing NearTax backend uses SQLite at `neartax.db`. For multi-user, we'll add a `users` table and scope all data by NEAR account ID.
</context>
