Tables to create/formalize:
1. ALTER users table: add columns `username VARCHAR(128) UNIQUE`, `email VARCHAR(256) UNIQUE`, `is_admin BOOLEAN DEFAULT FALSE`, `codename VARCHAR(64) UNIQUE` — all nullable (IF NOT EXISTS)
2. `passkeys` table: id UUID PK, user_id FK, credential_id TEXT UNIQUE NOT NULL, public_key BYTEA NOT NULL, counter BIGINT DEFAULT 0, device_type TEXT, backed_up BOOLEAN DEFAULT FALSE, last_used_at TIMESTAMPTZ, created_at TIMESTAMPTZ DEFAULT NOW()
3. `sessions` table: id TEXT PK (token_hex(32)), user_id FK, expires_at TIMESTAMPTZ NOT NULL, created_at TIMESTAMPTZ DEFAULT NOW(). Indexes on user_id and expires_at.
4. `challenges` table: id TEXT PK, challenge BYTEA NOT NULL, challenge_type TEXT NOT NULL ('registration','authentication','oauth_state','magic_link'), user_id FK nullable, expires_at TIMESTAMPTZ NOT NULL, metadata JSONB. Index on expires_at.
5. `magic_link_tokens` table: id TEXT PK, email TEXT NOT NULL, user_id FK nullable, expires_at TIMESTAMPTZ NOT NULL, used_at TIMESTAMPTZ nullable, created_at TIMESTAMPTZ DEFAULT NOW()
6. `accountant_access` table: id SERIAL PK, accountant_user_id FK, client_user_id FK, permission_level TEXT CHECK IN ('read','readwrite'), created_at TIMESTAMPTZ DEFAULT NOW(), UNIQUE(accountant_user_id, client_user_id)

Use `op.execute()` with raw SQL for IF NOT EXISTS patterns (same approach as migration 003). Add SQLAlchemy model classes for Passkey, Session, Challenge, MagicLinkToken, AccountantAccess to `db/models.py`.
