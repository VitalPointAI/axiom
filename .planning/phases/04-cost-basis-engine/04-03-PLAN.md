Create `engine/superficial.py`:

**SuperficialLossDetector class:**
- `__init__(self, conn)` — takes psycopg2 connection
- `scan_for_user(self, user_id: int) -> list[dict]`:
  Main method. Query capital_gains_ledger for all losses (gain_loss_cad < 0) for user_id. For each loss:
  1. Get token_symbol and block_timestamp from the ledger row
  2. Calculate 61-day window: disposal_ts - 30 days to disposal_ts + 30 days (in seconds)
  3. Query ALL acquisitions of same token in window across:
     - `transactions` table (on-chain): WHERE user_id=%s AND token_id matches token AND direction='in' AND block_timestamp BETWEEN window_start AND window_end
     - `exchange_transactions` table: WHERE user_id=%s AND asset=%s AND tx_type IN ('buy','receive') AND timestamp BETWEEN window_start AND window_end
  4. **Exclude** acquisitions from the same parent transaction (for swaps): filter out transaction_classifications with same parent_classification_id as the disposal's classification
  5. Sum total units rebought
  6. Calculate denied_ratio = min(1, total_rebought / units_disposed)
  7. denied_loss_cad = abs(gain_loss_cad) * denied_ratio
  8. Return list of superficial loss dicts: {ledger_id, token_symbol, gain_loss_cad, denied_loss_cad, denied_ratio, rebuy_count, needs_review: True}

- `apply_superficial_losses(self, user_id: int, losses: list[dict])`:
  For each superficial loss:
  1. UPDATE capital_gains_ledger SET is_superficial_loss=TRUE, denied_loss_cad=%s, needs_review=TRUE WHERE id=%s
  2. Find the corresponding ACB snapshot for the replacement units (first rebuy acquisition after disposal)
  3. UPDATE acb_snapshots: add denied_loss_cad to cost_cad_delta and recalculate total_cost_cad and acb_per_unit_cad for that snapshot
  4. Note: This is a proposal — specialist must confirm before finalizing (needs_review=True)

**Create `indexers/acb_handler.py`:**

```python
class ACBHandler:
    """Job handler for 'calculate_acb' job type.
    Registered in IndexerService. Triggered after classification completes."""

    def __init__(self, pool, price_service):
        self.pool = pool
        self.price_service = price_service

    def run_calculate_acb(self, job: dict) -> None:
        """Called by IndexerService when job_type == 'calculate_acb'."""
        user_id = job["user_id"]
        logger.info("Starting ACB calculation for user_id=%s", user_id)

        engine = ACBEngine(self.pool, self.price_service)
        stats = engine.calculate_for_user(user_id)

        logger.info(
            "ACB complete for user_id=%s: %d snapshots, %d gains, %d income, %d tokens, %d superficial losses",
            user_id,
            stats.get("snapshots_written", 0),
            stats.get("gains_recorded", 0),
            stats.get("income_recorded", 0),
            stats.get("tokens_processed", 0),
            stats.get("superficial_losses", 0),
        )
```
