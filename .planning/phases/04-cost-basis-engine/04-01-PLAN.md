**Extend PriceService (indexers/price_service.py)** with two new methods:

1. `get_price_at_timestamp(self, coin_id: str, unix_ts: int, currency: str = "usd") -> tuple[Optional[Decimal], bool]`:
   - Returns (price, is_estimated) tuple
   - Round unix_ts to nearest minute for cache key: `ts_minute = (unix_ts // 60) * 60`
   - Check price_cache_minute table first (SELECT WHERE coin_id, unix_ts=ts_minute, currency)
   - On cache miss: fetch CoinGecko `market_chart/range` with 2-hour window (unix_ts-3600 to unix_ts+3600)
   - Parse returned `prices` array: find closest timestamp to target (timestamps are in milliseconds from CoinGecko)
   - Set is_estimated=True if closest timestamp is more than 15 minutes (900 seconds) from target
   - Cache result in price_cache_minute via INSERT ON CONFLICT DO NOTHING
   - Respect existing _COINGECKO_DELAY rate limiting (use the _rate_limit_coingecko() pattern or equivalent delay)
   - Handle stablecoins: add STABLECOIN_MAP = {"tether": Decimal("1"), "usd-coin": Decimal("1"), "dai": Decimal("1")} — return (Decimal("1"), False) for stablecoins without API call
   - Use CoinGecko Pro base URL if API key exists

2. `get_boc_cad_rate(self, date_str: str) -> Optional[Decimal]`:
   - Fetch daily USD/CAD rate from Bank of Canada Valet API
   - URL: `https://www.bankofcanada.ca/valet/observations/FXUSDCAD/json?start_date={date_str}&end_date={date_str}`
   - No auth required
   - Parse response: `observations[0]["FXUSDCAD"]["v"]` -> Decimal
   - Cache in price_cache table as coin_id='usd', currency='cad', date=date_str, source='bank_of_canada'
   - Check cache first before API call
   - Weekend/holiday fallback: if no observation returned, look back 1 day at a time up to 5 days for most recent business day rate. Set source='bank_of_canada_fallback'
   - Return None if still no rate found after 5-day lookback

3. `get_price_cad_at_timestamp(self, coin_id: str, unix_ts: int) -> tuple[Optional[Decimal], bool]`:
   - Convenience method: get USD price at timestamp, then multiply by BoC CAD rate for that date
   - Return (price_cad, is_estimated) where is_estimated is True if either price or rate was estimated

Add constant: `BOC_VALET_BASE = "https://www.bankofcanada.ca/valet"`
