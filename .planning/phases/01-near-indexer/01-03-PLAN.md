<task type="auto">
  <name>Task 1: Build Staking History Indexer</name>
  <files>indexers/staking_rewards.py</files>
  <action>
Create staking rewards tracker:

1. Create `indexers/staking_rewards.py` with:
   - `index_staking_history(account_id)`:
     - Query NearBlocks for all transactions
     - Filter for staking-related actions:
       - `deposit_and_stake` to validator pools
       - `unstake` from validator pools
       - `withdraw` from validator pools
     - Insert into staking_events table:
       - wallet_id, validator_id, event_type, amount, block_timestamp, tx_hash
     - Return summary of deposits/withdrawals by validator
   
   - `get_current_staked_amount(account_id, validator_id)`:
     - Query validator pool contract via RPC
     - Get account's staked balance (includes rewards)
     - Return in NEAR
   
   - `calculate_staking_rewards(account_id)`:
     - Sum all deposit_and_stake events by validator
     - Sum all unstake/withdraw events by validator
     - Get current staked balance from pool
     - Rewards = current_staked + withdrawn - deposited
     - Return breakdown by validator

2. Handle multiple validators:
   - vitalpointai.near has staked with: vitalpoint.pool.near (current),
     bisontrails, epic, figment, lux, openshards, zavodil (historical)
   - Calculate total rewards across all validators

3. Output format:
   ```python
   {
     "account_id": "vitalpointai.near",
     "validators": [
       {
         "validator_id": "vitalpoint.pool.near",
         "total_deposited": 15000.0,
         "total_withdrawn": 0.0,
         "current_staked": 19716.0,
         "rewards": 4716.0
       }
     ],
     "total_rewards": 4716.0
   }
   ```
  </action>
