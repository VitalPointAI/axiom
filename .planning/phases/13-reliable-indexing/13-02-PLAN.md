  <action>
Create `indexers/near_stream_fetcher.py`:

1. Class `NearStreamFetcher(ChainFetcher)`:
   - __init__(self, pool, cost_tracker=None) — pool for DB, optional CostTracker for cost logging
   - NEARDATA_BASE = "https://mainnet.neardata.xyz" (from env NEARDATA_API_URL or default)
   - POLL_INTERVAL = 0.6 seconds

2. Core methods:
   - `async fetch_block(session, height)` — GET /v0/block/{height}, handle "null" response, return dict or None
   - `async get_last_final_block(session)` — GET /v0/last_block/final, return int height
   - `extract_wallet_txs(block, tracked_wallets: set)` — parse block JSON for receipts/transactions where signer_id or receiver_id is in tracked_wallets. Extract: tx_hash, signer_id, receiver_id, actions, block_height, block_timestamp. Use patterns from neardata_indexer.py for receipt parsing.
   - `async stream_blocks(start_height, tracked_wallets, on_tx_found)` — continuous poll loop: get last final block, fetch range [start+1, last], extract txs, call on_tx_found callback. Sleep POLL_INTERVAL between iterations.

3. ChainFetcher ABC implementation:
   - `sync_wallet(self, job)` — for historical sync (job_type full_sync/incremental_sync), delegate to NearBlocks API pattern from existing near_fetcher.py. Record last_block_height in json_cursor so streaming can pick up from there.
   - `get_balance(self, address)` — query NEAR RPC view_account for balance (or neardata.xyz if available)

4. Error handling:
   - Retry on HTTP 429/5xx with exponential backoff (max 60s)
   - Log and skip null blocks (neardata.xyz returns "null" for unavailable blocks)
   - If cost_tracker provided, wrap HTTP calls with cost_tracker.track('near', 'neardata_xyz', call_type)

5. Transaction upsert:
   - `upsert_transactions(self, pool, user_id, wallet_id, txs)` — INSERT INTO transactions ON CONFLICT DO NOTHING pattern (existing in near_fetcher.py). Reuse the same column mapping.

Write tests in `tests/test_near_stream_fetcher.py`:
- Mock aiohttp responses for block fetch (valid block, null block, 429 error)
- Test extract_wallet_txs with sample neardata.xyz block JSON (construct a realistic block fixture)
- Test stream_blocks with 3 mock blocks, verify callback called with correct txs
- Test that missing blocks (null response) are skipped without error
- Use unittest.mock.AsyncMock for async methods
