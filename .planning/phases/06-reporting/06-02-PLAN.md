       - Calls `_check_gate(user_id, tax_year)`
       - Computes fiscal year range via fiscal_year_range()
       - Queries acb_snapshots for peak total_cost_cad per token_symbol within fiscal year:
         ```sql
         SELECT token_symbol, MAX(total_cost_cad) AS peak_cost_cad
         FROM acb_snapshots
         WHERE user_id = %s
           AND TO_TIMESTAMP(block_timestamp) >= %s
           AND TO_TIMESTAMP(block_timestamp) <= %s
         GROUP BY token_symbol
         ```
       - Determines holding source per token: query wallets + transactions to see if token is held on exchange (exchange_transactions exists) or self-custody (NEAR/EVM wallet). Tokens on non-Canadian exchanges (Coinbase, Crypto.com, Uphold) = foreign property. Self-custodied = ambiguous.
       - CANADIAN_EXCHANGES = ['wealthsimple'] — Wealthsimple is Canadian; all others (coinbase, crypto_com, uphold, coinsquare) are considered foreign for T1135 purposes. Note: Coinsquare is technically Canadian but has US parent — flag for specialist review.
       - Sums peak_cost_cad for foreign-held tokens. Compares to $100,000 threshold.
       - Writes `t1135_check_{year}.csv`:
         Headers: Token, Peak Cost (CAD), Holding Source, Foreign Property, Notes
         Footer row with total foreign property cost and threshold determination
       - Returns dict: total_foreign_cost, threshold (100000), required (bool), tokens breakdown, self_custody_tokens (flagged ambiguous)

         Where Original Loss = gain_loss_cad (should be 0 after denial) and Denied Loss = denied_loss_cad
       - Returns dict: total_denied_cad, count, entries list
