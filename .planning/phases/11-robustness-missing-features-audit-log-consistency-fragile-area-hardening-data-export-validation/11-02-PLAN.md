  <name>Task 1: MANIFEST.json generation in PackageBuilder</name>
  <files>reports/generate.py, tests/test_reports.py</files>
  <behavior>
    - Test 1: PackageBuilder.build() creates MANIFEST.json in output_dir alongside report files
    - Test 2: MANIFEST.json contains "files" array with filename, sha256, size_bytes for each generated file
    - Test 3: MANIFEST.json does NOT include itself in the files array
    - Test 4: MANIFEST.json contains "source_data_version" with last_tx_timestamp, total_tx_count, acb_snapshot_version, needs_review_count
    - Test 5: MANIFEST.json contains generated_at (ISO 8601), tax_year, user_id
  </behavior>
  <action>
Add two private methods to PackageBuilder in reports/generate.py:

1. `_get_data_fingerprint(self, conn, user_id: int) -> dict`:
   - Query MAX(block_timestamp) from transactions WHERE user_id=%s
   - Query COUNT(*) from transactions WHERE user_id=%s
   - Query MAX(updated_at) from acb_snapshots WHERE user_id=%s
   - Query COUNT(*) from transaction_classifications WHERE user_id=%s AND needs_review=TRUE
   - Return dict with last_tx_timestamp, total_tx_count, acb_snapshot_version (isoformat if not None), needs_review_count
   - Also query exchange_transactions COUNT for user_id and add to total_tx_count

2. `_write_manifest(self, output_dir: str, user_id: int, tax_year: int, conn) -> str`:
   - Collect all files in output_dir using pathlib.Path(output_dir).glob("*") excluding MANIFEST.json itself
   - For each file: compute SHA-256 via hashlib.sha256(Path(f).read_bytes()).hexdigest(), get size_bytes via Path(f).stat().st_size
   - Build manifest dict with generated_at (datetime.utcnow().isoformat() + "Z"), tax_year, user_id, source_data_version (from _get_data_fingerprint), files array
   - Write to output_dir/MANIFEST.json with json.dumps(manifest, indent=2)
   - Return manifest_path

3. In build() method, after all report generation calls complete and before the return statement, call self._write_manifest(output_dir, user_id, tax_year, conn). This requires getting a conn from the pool — use self.pool.getconn()/putconn() pattern.

Add imports: hashlib, pathlib.Path (if not already imported).

Add tests in tests/test_reports.py for the manifest generation behavior described above. Use tmp_path fixture and mock the DB queries in _get_data_fingerprint.
