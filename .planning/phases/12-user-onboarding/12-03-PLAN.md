   - Display: Info-styled banner (bg-blue-900/30 border border-blue-800 rounded-lg p-4) with:
     - Icon (default: Info from lucide-react) + title (font-medium text-blue-300) + description (text-sm text-blue-200/70)
     - X close button (top right) that calls PATCH /api/preferences/dismiss-banner with `{ banner_key: bannerKey }` then hides via local state

   - 'use client' component
   - Props: `{ diagnosisCategory: string; verificationId: number; onAction?: () => void }`
   - Map diagnosis categories to plain-English explanations and actions:
     - `missing_staking_rewards`: "You sold or transferred tokens at a loss and appear to have missing staking rewards. Axiom detected a gap in your staking reward history." -> "Re-sync Staking" button -> POST /api/verification/resync/{verificationId}
     - `unindexed_period`: "There's a gap in your transaction history for this wallet. Some transactions may not have been indexed." -> "Re-index Wallet" button -> POST /api/verification/resync/{verificationId}

     - `uncounted_fees`: "Some network fees associated with this transaction may not be reflected in the cost basis calculation." -> "Mark Reviewed" button -> POST /api/verification/resolve/{verificationId}
   - Display: compact card below the verification issue row (bg-gray-800/50 border-l-2 border-yellow-500 pl-3 py-2)
   - Show explanation text (text-sm text-gray-300) + action button (small Button variant)
   - After action button click, show success feedback (checkmark + "Done") and call onAction callback
   - Handle unknown categories gracefully (show generic "This item needs review" text)
